Dieses Projekt ist ein LoRa Kommunikator welcher auf einem Raspberry Pi Zero 2w basiert. Er bietet Slots für i2c und one-Wire Sensoren(diese müssen in die Software eingebetet werden). Über den Button Sensor Logger können die Sensor daten im Hintergrund in einem bestimmten zeitinterval gespeichert werden und direkt auf der Grafischen Oberfläche ausgegeben werden.Die Funk Funktion bietet die mötglichkeit Textnachrichten an ein anderes Gerät in Reichweite zu senden. 


Part list:
        -Raspberry Pi Zero 2W
        -Ebyte E-220 900-22D
        -DS3231 ( for Raspberry)
        -MAX17048
